Unites In CSS

In CSS, units are used to define the size of elements, spacing, and other dimensions. There are two main types of units: absolute and relative.

Absolute Units

Absolute units include measurements like inches (in), centimeters (cm), and points (pt). These units are fixed and do not change based on the user's screen size or resolution.

Relative Units

Relative units include percentages (%), ems (em), and rems (rem). These units are flexible and can adapt to the user's screen size or settings.

Hello how are you (25.4mm)
Hello how are you (1in)
